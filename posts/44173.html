微软的 Windows 系统在市场上占了相当大的份额，同样，这个系统也因为其超高的兼容性和那堆成屎一样的代码而闻名于世（虽然我写的也好不到哪里去就对了），各种远古版本的特性到现在都没修，其美名曰是为了兼容性，但看起来应该是技术债（谁没有啊，我现在就看不懂以前写的朋友圈逻辑了）

自然，也会有许多开发者为 M$ 擦屁股。

这些软件有很多类型：优化系统、添加功能、修复 bug

¶前言

作为一位 Windows 用户，我对于 Windows 社区内部对于 Windows 系统魔改的软件持支持态度。这些软件可以优化 Windows 的使用体验，或者为 Windows 添加更加绚丽的 UI 或者桌面背景（wallpaper engine），更好的垃圾处理工具（C-cleaner），还有在系统支持周期结束后依旧在对系统提供维护的0patch（0patch！打钱！）

在开始以前，先猜猜我的电脑系统是WIN几?

这应该不是很难，但我的桌面像是win7和win10喝多了的产物，中间还夹杂着win11带来的更新（应该是更新吧）。坏事了，我好像说出来了，那就不管了，这就是windows 11 。

win11没有带来什么新的让我眼前一亮的更新，道有更多眼前一黑的东西，改版右键菜单、修改文件管理器、没有意义的云母效果、卡的要死的设置（好像10就有的问题）、几乎没用的电源设置（我现在还是用的PowerToys Awake吊着）

现在的微软三哥浓度有点太高了，连Windows这个集团最大的项目也能如此不上心，那自然可见其他项目有多么的水，整个公司都有一股草台班子的味道，现在维持WIN系统的似乎就只剩它（Windows 系统）积累的用户基数罢了。

接下来就是官方和社区的支持了。

¶👻Tai-监测软件

  • 👻 Tai

    • 开发者：Plan shit
    • Github: https://github.com/Planshit/Tai
    • 文档： 暂无
    • 免费软件

win本身没有一个好用的使用监测程序，但我本人也时常懊恼于：我的时间到底飞到哪里去了，这个时候就需要一个小程序看看你到底将时间飞到哪里去了，群友推荐了很多，但是我最后选择了 👻Tai ，表面上是为了不泄露隐私（应用程序声明了它们只在更新的时候产生网络流量），但那个小幽灵挺可爱的

根据 Github 上公开的信息，你在使用前要安装 .NET Framework 4.8，当然，假如你是WIN10及以上系统的用户可以直接忽略这段信息，（win已经预装了此运行库），在GitHub-Releases页面下载最新版本，完成后在下载目录找到👻Tai.exe ，启动程序，成功启动后你将在状态栏看到 👻 图标；

当然，你要记得你放在你哪里，因为👻Tai的卸载有点原始：

  1. 在设置中停用开机自启动（如果启用了）；

  2. 删除所有文件即可完成卸载。

它提供了浏览器插件用来监视（统计）你的浏览器使用情况。目前，可以统计网站的名字和访问时间（上一张图片的应用边上的网站）其中的数据就不用我解释了吧，如果不想被查岗的话，别让这个东西被你的老板发现！

应用程序的数据存储在[tai目录]/data/data.db下面，你随时可以作死删除掉这个文件，反正我是不会去进行这种尝试，目录里面还有一个AppConfig.json不知道是干什么的，就先不管了

当然，它还有其他非常强大的功能等着我们发现和使用。就类似于：关联进程(多屏工作时将其他屏幕所使用的软件同时记录使用时长)。但是，碍于篇幅有限（主要是我也没有研究透）我们暂时按下不表，等各位大佬摸清楚这个统计程序的时，记得踢我一下

¶StartAllBack-美化工具

  • StartAllBack

    • 开发者：未知
    • 官网：https://www.startallback.com/
    • github：闭源软件
    • 收费软件（4$/设备）

注意
使用bing国内版搜索出来的第一个结果无法确认是否为官方站点，如要访问官方站点，认准：startallback.com

win11自带的快捷栏和资源管理器很难让人恭维。而且，M$ 的更新还将我一直在使用的 EarTrumpet 一个 Windows 的音量控制组件直接搞不见（找不到入口）了，这让我非常不爽，同时，win11的更新也将文件资源管理器改得一塌糊涂（什么反向升级），于是，在一次互联网检索中，我遇到了StartAllBack。

StartAllBack 本身的功能异常强大，可以对任务栏和资源管理器进行各种各样的骚操作，包括但不限于：

  • 修改任务栏

  • 使用旧版文件资源管理器顶部栏样式

  • 修改通知中心和时间界面

  • 启用和禁用托盘图标

就博客开头的那张桌面截图就是我调过设置后截的图

对于任务栏的win开始徽标，你可以在致美化找到更多的开始按钮素材。当然，你还可以在致美化上下载鼠标样式、动态背景、还有第三方主题

StartAllBack 的功能非常好，就一点要注意：它是付费的，如果你要支持正版的话要向开发者支付 4$ /设备，对于我来说有点小贵了，毕竟学生吗，囊中羞涩很正常吧。（当然，你可以在伟大的简中互联网找到免费版，就是安全性，不敢保证。）

¶Dism++ 系统优化工具

  • Dism++

    • 开发者：Chuyu Team（团队）
    • Github: https://github.com/Chuyu-Team/Dism-Multi-language
    • 文档：未知
    • 免费软件

此工具包括在图吧工具箱 2023 内

Dism++由大学时期几个自症晚期患者打造，追求强功能、极致绿色的极小身材。 Dism++ 可以说是一个 Dism 的GUI 版，但是并不依赖 Dism，接基于更底层的 CBS(Component Based Servicing Reference)。 因此 Dism++往往比微软自带的 Dism 有更好的兼容性以及更加轻巧的体积。

其提供了相对强大的功能，能满足非专业用户（像我这样的）和专业用户的使用需求，而我主要使用的是：空间回收和系统优化。其他的页面要么是不懂，要么是不敢乱尝试

空间回收可以从系统底层回收被系统占用的空间，这也是我最喜欢的功能（主要是有空间焦虑）

除了空间回收，Dism++还提供了系统优化，这个部分主要是各种各样的开关，控制系统各种各样的功能的开启与关闭，提供了各种各样的使用功能，包括但不限于：禁用Windows更新、创建快捷方式时不添“快捷方式”文字、隐藏快捷方式小箭头、隐藏科执行文件小盾牌等功能，（可以不用去改注册表了捏🤣）

此外，在系统工具箱还提供：

  • 春哥附体

    • 使程序无视权限运行

  • 系统备份

  • 系统还原

  • 账户管理

    • 管理系统上面的账户

程序是免费的，非常良心，但是你无法在开源平台找到它的源代码，这算是开源了吗？如开（开了，有没有完全开）。但是好用是没得说的。

接下来我会介绍一下我用的较多的小功能：

¶PowerToys.Run-快速启动

这是一个非常好用的小软件，它的外观Utools更加圆滑，同时也足够轻量、齐全，平时用来启动应用程序就不用在到桌面去一个个找了，轻轻Alt+Space就可以轻松唤出来进行快捷的运行和启动

它支持实时扫描正在运行的程序，同时，基于Windows索引的文件检测器可以让你使用?查找文件和使用.访问程序，还支持诸如：哈希运算、单位换算、互联网搜索、运行Windows命令等，功能足够强大（是我喜欢的料）

¶PowerToys.AlwaysOnTop-强制置顶

我的电脑屏幕有点小，假如我要调某些东西，想一直看到那个窗口，但是有会被其他窗口覆盖且软件本身没有置顶程序，我推荐你使用这个，它可以将任何一个应用的窗口强制置顶，在小屏幕设备上使用类似计算器还有某些编辑器都会享受到这个功能带来的福利。

毕竟不用不同窗口切来切去。

¶PTs.KeyboardManagerEngine-键盘管理器

当初认识 PowerToys 就是应为自己玩的一款游戏它不支持修改键位。就充分发挥主观能动性，去搜：键盘映射工具，接着，就搜到了PowerToys的头上来，接触了PowerToys和它的键盘映射器。

其实这个功能非常的简单，就是用一个虚拟键盘输入，正式键盘的键位映射到虚拟键盘上面，就像一条条先一样.按下A，虚拟键盘就发送：Ctrl+A诸如此类。

你可以使用它来：

  • 简化操作

  • 修改游戏键位

  • 创建老板键

¶PowerToys.Awake-强制唤醒

这个工具旨在使计算机保持清醒状态，而无需管理其电源和睡眠设置。运行耗时较长的任务时，它非常有用，可确保计算机不会进入睡眠状态或关闭其屏幕。

¶还有更多！

PowerToys还有其他的功能，比如：改host文件的 PowerToys.HostFileEditor 、快速批量重命名的 PowerToys.PowerRename 、提供类似360文件占用解除的PowerToys.FileLocksmith、让Windows拥有像MacOS那样快速预览的PowerToys.Peek等等，这些应用程序可以解决和方便Windows的问题和使用。

PowerToys是免费的开源软件，啊！开源盛世，你可以点击这里前往Github下载PowerToys的最新版本
